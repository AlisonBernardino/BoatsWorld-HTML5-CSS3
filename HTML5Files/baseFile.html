Using "h2" and "hr" tags

We use a "h2" tag to target a Header type 2 text (Mostly known as model 02 title)

"hr" tags are used to make a black line like the example line above this text (This text line was also typed inside a "p" tag)

The "br" tag

A tag known as "Break line" tag and used to break a text in 02 (two) parts.
This part is being show after a "break" (or "br" tag) insertion inside the text.

Observation: To see the tags working, all the tags should be typed between "<" and ">" symbols (Without double quotes)
Examples:
< br> (Without the blank spaces between the tag and the symbols)
< hr>

Adding special symbols

We can add special symbols to our text, bars, items or components inside the HTML file.

Here are some examples:

  • ® = We type the "&" symbol + "reg" word
  • © = "&" + "copy"
  • ™ = "&" + "#8482"
  • ¢ = "&" + "cent"
  • £ = "&" + "pound"
  • ¥ = "&" + "yen"
  • Δ = "&" + "#916"
  • € = "&" + "#8364"

Observation: To see the results, all codes should be typed without "double quotes".
