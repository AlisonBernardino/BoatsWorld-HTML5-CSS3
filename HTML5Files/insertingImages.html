FrontEnd World - Inserting images (The "img" tag)

To insert images on the website, we use
the "img" tag (We type < img> )

After this point, we need to define the items known
as "tag parameters".

We will use 02 parameters:

  • "src" item
  • "alt" item

Their meanings are:

  • "src" = Used to insert the image location inside the project files.
  • "alt" = Alternative text to be shown after image presentation malfunction.
    If we suffer an image presentation bug/error, this text will appear.
